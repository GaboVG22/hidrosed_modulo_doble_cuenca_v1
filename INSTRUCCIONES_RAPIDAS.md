# Instrucciones rápidas

1. Abrir la carpeta del módulo.
2. Instalar dependencias con `pip install -r requirements.txt`.
3. Ejecutar `streamlit run app.py`.
4. Cargar un DEM GeoTIFF común o usar descarga OpenTopography.
5. Cargar dos puntos:
   - PC-HIDRO.
   - PC-DESCARGA.
6. Ejecutar doble delimitación.
7. Revisar:
   - área PC-HIDRO;
   - área PC-DESCARGA;
   - intercuenca;
   - advertencias de contención;
   - caudal total.
8. Descargar ZIP completo.
